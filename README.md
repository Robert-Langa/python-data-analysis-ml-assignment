# Python Data Analytics & Machine Learning Project

This project was completed as part of the Data Analytics, AI & Machine Learning course at Conestoga College.

The goal of the assignment was to apply Python programming to perform data manipulation, statistical analysis, visualization, and introductory machine learning using industry-standard data science libraries.

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn

## Project Overview

This project demonstrates several important data science and machine learning concepts:

• Data manipulation using NumPy  
• Data analysis and processing using Pandas  
• Data visualization using Matplotlib  
• Machine learning classification using Scikit-Learn  
• Statistical analysis and probability calculations  
• Data normalization techniques used in machine learning pipelines  

## Implemented Tasks

### Machine Learning
- Implemented a **k-Nearest Neighbours (k-NN) classifier** to predict iris flower species.
- Compared model performance for different values of **k (1, 5, 15)**.
- Evaluated model accuracy using a **train-test split**.

### Numerical Computing (NumPy)
- Created arrays and performed vectorized mathematical operations.
- Reshaped arrays and performed element-wise transformations.
- Computed row and column statistics efficiently.

### Data Visualization (Matplotlib)
- Plotted polar coordinate functions.
- Generated multiple Gaussian distribution curves.
- Visualized trends using graphs and labelled axes.

### Data Analysis (Pandas)
- Processed census population data to analyze demographic trends.
- Validated the **“Silver Tsunami” aging population hypothesis**.
- Performed cross-tabulation analysis on dataset categories.
- Generated HTML summary tables for reporting.

### Sports Analytics
- Implemented the **Pythagorean Theorem of Baseball** model.
- Calculated predicted vs actual winning percentages.
- Evaluated model performance using **Mean Absolute Deviation (MAD)**.

### Probability & Statistics
- Computed empirical probabilities from dataset observations.

### Data Normalization
- Implemented **z-score normalization** for machine learning preprocessing.
- Scaled dataset features using statistical transformation.
